## Frame Format Reference (from firmware/esp32-csi-node/main/csi_collector.c)

ADR-018 binary frame (little-endian):

| Offset | Size | Field |
|--------|------|-------|
| 0 | 4 | Magic: 0xC5110001 |
| 4 | 1 | Node ID |
| 5 | 1 | Antennas |
| 6 | 2 | Subcarriers |
| 8 | 4 | Frequency MHz |
| 12 | 4 | Sequence |
| 16 | 1 | RSSI |
| 17 | 1 | Noise floor |
| 18 | 2 | Reserved |
| 20 | 104 | I/Q data (52 subcarriers × 2 bytes: int8 I, int8 Q) |

**Total frame size:** 20 (header) + 104 (I/Q) = 124 bytes

**Context note:** The CONTEXT.md mentions 121 bytes, but the actual firmware serializes 52 subcarriers × 2 = 104 I/Q bytes + 20 header = 124 bytes. Use 124 bytes as the validation length, and extract `n_subcarriers` from the header (should be 52). If `n_subcarriers` differs, compute expected length as `20 + n_subcarriers * 2` and validate against that.

**Amplitude:** `sqrt(I^2 + Q^2)` for each subcarrier
**Phase:** `atan2(Q, I)` for each subcarrier (radians)

    1. Create `aggregator/parser.py`.
    2. Import `struct`, `math`, `logging`, and `CSIFrame` from `.frame`.
    3. Define `CSI_MAGIC = 0xC5110001` and `CSI_HEADER_SIZE = 20`.
    4. Implement `parse_frame(data: bytes) -> CSIFrame | None`:
       - If `len(data) < CSI_HEADER_SIZE`: return None
       - Unpack magic with `struct.unpack('<I', data[0:4])[0]`
       - If magic != CSI_MAGIC: log warning "Invalid magic ... dropping frame", return None
       - Unpack node_id: `data[4]`
       - Unpack antennas: `data[5]`
       - Unpack n_subcarriers: `struct.unpack('<H', data[6:8])[0]`
       - Unpack frequency_mhz: `struct.unpack('<I', data[8:12])[0]`
       - Unpack sequence: `struct.unpack('<I', data[12:16])[0]`
       - Unpack rssi: `struct.unpack('b', bytes([data[16]]))[0]`
       - Unpack noise_floor: `struct.unpack('b', bytes([data[17]]))[0]`
       - Compute expected_len = CSI_HEADER_SIZE + n_subcarriers * 2
       - If `len(data) != expected_len`: log warning "Frame length mismatch ...", return None
       - Iterate i from 0 to n_subcarriers-1:
         - I = `struct.unpack('b', bytes([data[CSI_HEADER_SIZE + i*2]]))[0]`
         - Q = `struct.unpack('b', bytes([data[CSI_HEADER_SIZE + i*2 + 1]]))[0]`
         - amplitude = `math.sqrt(I*I + Q*Q)`
         - phase = `math.atan2(Q, I)`
       - Return `CSIFrame(...)` with all fields
    5. Use `logging.getLogger(__name__)` for warnings.
    6. Wrap all unpacking in try/except; on any exception log warning and return None (per D-09: never crash).
