## Key Decisions to Implement
- **D-10:** Dynamic node discovery — no hardcoded node list. Auto-register on first valid frame. Node table is `dict[node_id, NodeState]`. Mark stale after 10s of no frames, retain slot.
- **D-06:** Asyncio Queue as inter-phase handoff — parsed frames pushed into `asyncio.Queue`. Phase 3 consumer pulls from queue.
- **D-07:** Drop Oldest (ring buffer) per node, default 500 frames (~10s @ 50Hz). When full, oldest dropped.

## Frame Rate Logging
- Log frame rate per node every 1 second using a recurring `asyncio` task.
- Detect packet loss via sequence gaps: if `current_seq - last_seq > 1`, log `dropped = current_seq - last_seq - 1`.
- Alert if sustained loss > 5% over 10-second window.

## Existing Prototype Reference
- `firmware/esp32-csi-node/test/web_aggregator.py` — threading-based UDP listener (NOT asyncio; do not copy)
- `firmware/esp32-csi-node/test/test_udp.py` — basic socket recvfrom
