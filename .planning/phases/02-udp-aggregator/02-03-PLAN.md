## Key Decisions to Implement
- **D-08:** Persist raw CSI to `.npy` files for Phase 5 dataset collection
  - Format: NumPy binary `.npy` with shape `(frames, 52)` for amplitudes
  - One file per recording session; auto-rotated by timestamp
  - Metadata JSON alongside each `.npy` (node_id, timestamps, labels if available)
  - Directory structure: `data/raw/YYYY-MM-DD_HH-MM/`
- **D-06:** Asyncio Queue as inter-phase handoff — main.py runs both producer (server) and consumer (persistence + Phase 3 queue) in same event loop

## Existing Prototype Reference
- `llm-wiki/raw/RuView/scripts/record-csi-udp.py` — JSONL writer pattern (do NOT copy JSONL format; use `.npy` per D-08)
- `llm-wiki/raw/wallhack1.8k/datasets.py` — amplitude matrix shape `(frames, 52)` consumed by PyTorch

## Downstream Contract
Phase 5 expects `.npy` files with shape `(frames, 52)` containing amplitude values. The `NpyWriter` must accumulate frames and flush them to disk on rotation or shutdown.

       - `__init__(self, output_dir: str = "data/raw", rotation_frames: int = 10000)`:
         - `self.output_dir = pathlib.Path(output_dir)`
         - `self.rotation_frames = rotation_frames`
         - `self._buffers: dict[int, list[list[float]]] = {}`  # node_id -> list of amplitude vectors
         - `self._frame_counts: dict[int, int] = {}`
         - `self._start_times: dict[int, float] = {}`
         - `self._session_dir: pathlib.Path | None = None`
       - `_ensure_session_dir(self) -> pathlib.Path`:
         - If `self._session_dir` is None: create `self.output_dir / datetime.now().strftime("%Y-%m-%d_%H-%M")`
         - Return the path
       - `write(self, frame: CSIFrame) -> None`:
         - `node_id = frame.node_id`
         - If node_id not in `_buffers`: initialize empty list, set `_start_times[node_id] = time.time()`, `_frame_counts[node_id] = 0`
         - Append `frame.amplitudes` to `_buffers[node_id]`
         - Increment `_frame_counts[node_id]`
         - If `_frame_counts[node_id] >= self.rotation_frames`: call `_flush_node(node_id)`
       - `_flush_node(self, node_id: int) -> None`:
         - If node_id not in `_buffers` or empty: return
         - `session_dir = self._ensure_session_dir()`
         - `timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")`
         - `npy_path = session_dir / f"node_{node_id}_{timestamp}.npy"`
         - `meta_path = session_dir / f"node_{node_id}_{timestamp}.json"`
         - `arr = np.array(self._buffers[node_id], dtype=np.float32)`  # shape (N, 52)
         - `np.save(npy_path, arr)`
         - Write metadata JSON: `{"node_id": node_id, "start_time": self._start_times[node_id], "frame_count": len(self._buffers[node_id]), "shape": list(arr.shape)}`
         - Clear `_buffers[node_id]` and reset `_frame_counts[node_id] = 0`
         - Log at INFO: "Flushed {n} frames to {npy_path}"
       - `flush_all(self) -> None`:
         - For each node_id in `_buffers`: call `_flush_node(node_id)`
    4. Use `np.float32` for compact storage (half the size of float64).
    5. Do NOT write phase data to `.npy` — only amplitudes per D-08 and Phase 5 needs.
