<task type="auto">
  <name>Task 2: Implement Hampel outlier filter</name>
  <files>processor/hampel.py, tests/test_hampel.py</files>
  <action>
    1. Create `processor/hampel.py` with function:
       - `hampel_filter(data: np.ndarray, window_size: int = 7, threshold: float = 3.0) -> np.ndarray`
       - Input: 1D numpy array of amplitude values for a single subcarrier over time
       - Algorithm (per D-15, pure numpy, no scipy):
         - For each index i, define window = data[max(0, i-half):min(n, i+half+1)] where half = window_size // 2
         - Compute median = np.median(window)
         - Compute MAD = np.median(np.abs(window - median)) * 1.4826 (MAD to std conversion)
         - If |data[i] - median| > threshold * MAD, replace data[i] with median
       - Return cleaned 1D array (same length, same dtype)
    2. Handle edge cases: window_size must be odd (if even, add 1), MAD == 0 (skip replacement to avoid division by zero)
    3. Create `tests/test_hampel.py` with tests:
       - Synthetic spike: array of 1.0 with single spike to 100.0 → spike should be replaced with ~1.0
       - Verify >80% amplitude reduction on spike (D-15 success criterion)
       - No false positives: clean Gaussian noise (σ=0.1) should have <5% replacements
       - Edge cases: window_size=3, single-element array, all-constant array
  </action>
