  <name>Task 1: Create CsiProcessor with asyncio task and per-node state</name>
  <files>processor/main.py</files>
  <action>
    1. Create `processor/main.py` with class `CsiProcessor`:
       - `__init__(self, input_queue: asyncio.Queue, output_queue: asyncio.Queue | None = None, config: dict | None = None)`:
         - Store input_queue (from aggregator, D-06) and output_queue (for Phase 4, D-13)
         - Config dict with defaults per D-12:
           ```python
           {
             "window_size": 200,      # 4s @ 50 Hz
             "step_size": 100,        # 50% overlap
             "n_subcarriers": 64,
             "sample_rate": 50.0,
             "hampel_window": 7,
             "hampel_threshold": 3.0,
           }
           ```
         - Merge provided config over defaults
         - Internal state: `self._windows: dict[int, SlidingWindow]` — one per node_id (D-18)
         - Shutdown event: `self._shutdown = asyncio.Event()`
       - `async def run(self) -> None`:
         - Loop until shutdown event is set:
           - `frame = await asyncio.wait_for(self.input_queue.get(), timeout=1.0)`
           - Call `self.process_frame(frame)`
         - On asyncio.TimeoutError: continue (allows checking shutdown event)
         - On asyncio.CancelledError: break cleanly
       - `process_frame(self, frame: CSIFrame) -> dict | None`:
         - Get or create SlidingWindow for frame.node_id
         - Extract amplitudes as np.array (shape 64,)
         - Optionally extract phases as np.array (shape 64,) for presence detection path
         - Push amplitudes into SlidingWindow
         - If window emits (returns non-None):
           - Apply Hampel filter per subcarrier: for each column in window, call `hampel_filter(column)`
           - Optionally: unwrap + detrend phase window if phases were collected
           - Call `extract_features(cleaned_amplitudes, cleaned_phases)`
           - Build output dict per D-16 with metadata:
             ```python
             {
               "node_id": frame.node_id,
               "window_start_ms": int,   # timestamp of first frame in window
               "window_end_ms": int,     # timestamp of last frame in window
               "features": np.ndarray,   # flat array of shape (n_features,)
             }
             ```
           - If output_queue is set: `await self.output_queue.put(feature_dict)` (D-13)
           - Return feature_dict
         - If window not full: return None
       - `stop(self) -> None`: Set shutdown event
    2. Handle edge cases:
       - Frame with wrong n_subcarriers: log warning, skip
       - Node goes stale (no frames for 10s): log info, keep window state (don't reset — may resume)
    3. Add module-level logger
  </action>
