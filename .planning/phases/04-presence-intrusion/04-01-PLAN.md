```python
feature_dict = {
    "node_id": int,
    "window_start_ms": int,
    "window_end_ms": int,
    "features": np.ndarray,  # 1D float64, shape (N*2 + 2,)
}
```
Flat array layout (N = n_subcarriers, typically 64):
- indices [0:N]      → mean_amp per subcarrier
- indices [N:2*N]    → var_amp per subcarrier
- index [2*N]        → motion_energy (scalar)
- index [2*N + 1]    → breathing_band (scalar)

N can be computed as: `(len(flat) - 2) // 2`

2. `PresenceDetector` class with signature:
   ```python
   class PresenceDetector:
       def __init__(
           self,
           node_id: int,
           enter_threshold_sigma: float = 2.5,   # D-24
           exit_threshold_sigma: float = 1.5,    # D-24
           enter_frames: int = 3,                # D-25
           exit_frames: int = 5,                 # D-25 (≈10s at 0.5 Hz feature rate)
           baseline_alpha: float = 0.1,          # EMA learning rate (agent discretion)
           min_baseline_frames: int = 10,        # frames before baseline is valid
       ) -> None:
   ```

3. `update(self, feature_vector: np.ndarray) -> dict | None` method:
   - Parse flat array: N = (len(fv) - 2) // 2; motion = fv[2*N]; breathing = fv[2*N + 1]
   - Compute combined score: `score = 0.7 * motion + 0.3 * breathing` (D-22 weighted)
   - If baseline not ready (frame_count < min_baseline_frames): use score to build baseline
   - Baseline tracking (D-21): EMA update of mean and std of `score` when state is EMPTY or CONFIRMING_EMPTY
     ```python
     self._baseline_mean = (1-alpha)*self._baseline_mean + alpha*score
     self._baseline_std = (1-alpha)*self._baseline_std + alpha*abs(score - self._baseline_mean)
     ```
   - Deviation: `sigma = (score - baseline_mean) / max(baseline_std, 1e-6)`
   - State machine transitions (D-24, D-25):
     - EMPTY → CONFIRMING_OCCUPIED if sigma > enter_threshold_sigma
     - CONFIRMING_OCCUPIED → OCCUPIED after `enter_frames` consecutive triggers
     - CONFIRMING_OCCUPIED → EMPTY if sigma drops below enter_threshold (reset counter)
     - OCCUPIED → CONFIRMING_EMPTY if sigma < exit_threshold_sigma
     - CONFIRMING_EMPTY → EMPTY after `exit_frames` consecutive below-threshold frames
     - CONFIRMING_EMPTY → OCCUPIED if sigma rises above exit_threshold (reset counter)
   - Return dict only on state transitions (not every frame):
     ```python
     {
         "node_id": self.node_id,
         "status": state.value,        # "empty" | "occupied"
         "confidence": float,          # sigmoid(abs(sigma) / 5.0) clamped 0.0-1.0
         "trigger_feature": "combined",
         "sigma": float,               # raw deviation for debugging
     }
     ```
   - Return `None` if no state transition occurred
   - Input validation: if `np.any(np.isnan(fv))` or `np.any(np.isinf(fv))`, log warning and return None

4. `reset_baseline(self) -> None` — force re-learning (useful for testing)

5. `is_baseline_ready(self) -> bool` property
