    Create classifier/augment.py implementing D-43 augmentation requirements:
    
    1. **shift_augment(X, y, max_shift=10)**:
       - For each sample, create shifted versions by rolling along time axis: np.roll(X, shift, axis=1)
       - Shifts: range(-10, 11) excluding 0 = 20 shifts total (per D-43: ±10 steps, 20×)
       - Concatenate all shifted versions with original data
       - Labels are duplicated for each shift
       - Returns augmented X, y with shape (samples*21, time_steps, subcarriers)
    
    2. **noise_augment(X, y, num_copies=3, noise_std=1.0)**:
       - Multiplicative Gaussian noise: X_noisy = X + X * noise where noise ~ N(0, noise_std)
       - Creates num_copies (default 3) noisy versions per D-43
       - Concatenate noisy versions with original data
       - Labels duplicated for each copy
       - Returns augmented X, y with shape (samples*4, time_steps, subcarriers)
    
    3. **mixup_augment(X, y, alpha=1.0, prob=0.3)**:
       - Randomly select pairs of samples with probability 0.3 (per D-43)
       - Mix: λ ~ Beta(α, α), X_mix = λ*X_i + (1-λ)*X_j
       - Label mixing for one-hot labels: y_mix = λ*y_i + (1-λ)*y_j
       - Returns mixed batch (same shape as input)
    
    4. **augment_dataset(dataset, shift=True, noise=True, mixup=False)**:
       - Convenience function that applies augmentation to a dataset
       - Extracts all X, y from dataset, applies augmentation, returns new dataset
       - Default: shift + noise (MixUp optional per D-43)
    
    All functions operate on numpy arrays (not torch tensors) to avoid GPU memory issues with large augmented datasets.
    Ensure deterministic behavior with fixed random seed for reproducibility.
