  <action>
    Create classifier/infer.py with real-time inference infrastructure:
    
    1. **ActivityLabel dataclass**:
       - Fields: timestamp (ISO8601), node_id (int), label (str), confidence (float), class_probs (dict)
       - to_dict() method for Queue serialization
    
    2. **SlidingWindowBuffer** (internal class):
       - Per-node circular buffer for accumulating raw amplitude frames
       - Window size: 50 frames (per D-37)
       - Step size: 25 frames (50% overlap, per D-37)
       - When window is full, emits (node_id, window_array) where window_array shape is (50, subcarriers)
       - Center-crops/pads subcarriers to 52 (same as training per D-36)
       - Handles variable subcarrier counts using same crop/pad logic as dataset.py
    
    3. **CsiClassifier** class:
       - __init__(input_queue, output_queue, model_path, scaler_path, config):
         - input_queue: asyncio.Queue receiving raw frames (same format as processor input)
         - output_queue: asyncio.Queue emitting ActivityLabel dicts
         - model_path: path to trained .pth checkpoint
         - scaler_path: path to scaler JSON
         - config: dict with confidence_threshold (default 0.5, optional per discretion)
       - Loads model from checkpoint using classifier.train.load_checkpoint
       - Loads StandardScaler from JSON
       - Sets model to eval mode
       - Tracks per-node SlidingWindowBuffer
       - max_nodes: 16 (same as detector)
    
    4. **run() method** (asyncio task):
       - While not shutdown:
         - Wait for frame from input_queue (timeout 1.0s for stale checks)
         - Extract amplitude from frame (sqrt(I^2 + Q^2) — reuses parser logic or receives pre-computed amplitude)
         - Append to node's SlidingWindowBuffer
         - When buffer emits a full window:
           - Normalize with StandardScaler (reshape to 2D, transform, reshape back)
           - Convert to torch tensor (1, 50, 52)
           - Run model forward pass (with torch.no_grad())
           - Apply softmax to get class probabilities
           - Select class with max probability as label
           - If confidence < threshold (optional), label = "unknown"
           - Create ActivityLabel and emit to output_queue
       - Graceful shutdown on cancellation
    
    5. **stop() method**:
       - Sets shutdown event
    
    6. **Performance target**:
       - Log inference time per window
       - Assert <10ms per sample (per ACT-05)
    
    Label mapping: 0=walking, 1=running, 2=lying, 3=bending
