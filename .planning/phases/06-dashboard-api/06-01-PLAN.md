    - `__init__(self, alert_queue, activity_queue, amplitude_queue, node_source=None)` where node_source is a callable returning dict[int, NodeState] (e.g., `server.nodes`)
    - Maintain internal state:
      - `self._presence: dict[int, str]` — last known presence status per node (default "unknown")
      - `self._activity: dict[int, dict]` — last known ActivityLabel.to_dict() per node
      - `self._alerts: deque[dict]` — recent alerts, `deque(maxlen=50)`
      - `self._heatmap: dict[int, deque[np.ndarray]]` — recent amplitude arrays per node, `deque(maxlen=200)` (200 frames ≈ 20s at 10 fps)
      - `self._shutdown: asyncio.Event`
    - `async def start(self)` — creates three consumer tasks:
      - `_consume_alerts()`: `while True: item = await self.alert_queue.get(); if isinstance(item, dict): self._alerts.appendleft(item); elif hasattr(item, 'to_dict'): self._alerts.appendleft(item.to_dict()); if item.get('type') == 'intrusion' or getattr(item, 'type', None) == 'intrusion': self._presence[item.get('node_id', getattr(item, 'node_id', 0))] = item.get('status', getattr(item, 'status', 'unknown'))`
      - `_consume_activity()`: `while True: item = await self.activity_queue.get(); self._activity[item.get('node_id', getattr(item, 'node_id', 0))] = item if isinstance(item, dict) else item.to_dict()`
      - `_consume_amplitude()`: `while True: frame = await self.amplitude_queue.get(); node_id = getattr(frame, 'node_id', frame.get('node_id')); amps = getattr(frame, 'amplitudes', frame.get('amplitudes')); if node_id is not None and amps is not None: self._heatmap.setdefault(node_id, deque(maxlen=200)).append(np.asarray(amps))`
    - `def stop(self)` — sets `self._shutdown.set()`, then cancels all three consumer tasks and awaits them with `asyncio.gather(..., return_exceptions=True)`
    - `def get_status(self) -> dict` — returns `{"presence": self._presence.copy(), "activity": self._activity.copy(), "node_health": self._get_node_health()}`
    - `def get_alerts(self, count: int = 50) -> list[dict]` — returns `list(self._alerts)[:count]`
    - `def get_heatmap(self) -> dict[int, list[list[float]]]` — converts each node's deque to a list of lists (serializable), e.g., `{nid: [arr.tolist() for arr in self._heatmap[nid]] for nid in self._heatmap}`
    - `def _get_node_health(self) -> dict` — if `self._node_source` is not None, reads `self._node_source()` and returns `{nid: {"fps": ... , "stale": node.stale, "frame_count": node.frame_count, "last_seen": node.last_seen} for nid, node in nodes.items()}`; otherwise empty dict

    Use `logging.getLogger("dashboard.state")` for diagnostics. No print statements.
